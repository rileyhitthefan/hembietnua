# PartyRock labs (no AWS access required)

#### Overall
 [**PartyRock**](https://partyrock.aws/), an Amazon Bedrock playground, is a space where you can build and use AI-generated apps in a playground powered by [**Amazon Bedrock**](https://aws.amazon.com/bedrock/). Itâ€™s a fast and fun way to learn about generative AI. It does not require an AWS account to get started. Anyone can experiment with PartyRock by creating a profile using a social login from Amazon.com, Apple, or Google.
 
 These labs assume no previous experience with generative AI, writing code, or app development. You will use PartyRock's capabilities to create several different generative AI applications.
